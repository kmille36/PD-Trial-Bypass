# PD-Trial-Bypass
Simple Bypass Parallels' trial in Mac OS! Work for ALL versions!

- Install:
  ```sh
  sudo curl -sLk -o /usr/local/bin/pd-clean https://github.com/kmille36/PD-Trial-Bypass/raw/refs/heads/main/pd-clean && sudo chmod +x /usr/local/bin/pd-clean
  ```

- Remove:
  ```sh
  sudo rm -rf /usr/local/bin/pd-clean
  ```

- Usage:
  ```sh
  pd-clean
  ```
