# PD-Trial-Bypass
Simple Bypass Parallels' trial in Mac OS! Work for ALL versions!

- Install:
  sudo curl -sLk -o /usr/local/bin/pd-clean https://github.com/kmille36/PD-Trial-Bypass/raw/refs/heads/main/pd-clean && chmod +x /usr/local/bin/pd-clean

- Remove:
  sudo rm -rf /usr/local/bin/pd-clean

- Usage:
  sudo pd-clean
